
# vCard App with REACT NATIVE (EXPO)

vCard App Project make with REACT NATIVE (EXPO)
using JavaScript

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Install expo package

```bash
 npm add expo
```

Start the server

```bash
  npx expo start
```
